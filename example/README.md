# common_utils Example

This Flutter project demonstrates the various features of the `common_utils` package.

## Getting Started

This example app is a simple Flutter application that showcases the different utilities available in the parent package. To run the example:

1.  Navigate to the `example` directory in your terminal.
2.  Run `flutter pub get` to install the dependencies.
3.  Run `flutter run` to launch the application on a connected device or simulator.

## Examples

The app provides a simple navigation menu to access different examples:

*   **Theming Example:** Demonstrates the use of `AppSpacer`, `SoftGrid`, and `AppTypography` for creating consistent and well-structured layouts.
*   **Validation Example:** Shows how to use the `FormValidator` class to validate user input in a form.
*   **Other Utilities Example:** Covers the usage of `BuildContextShortcuts`, `DebugLogger`, and `PlatformUtils`.
